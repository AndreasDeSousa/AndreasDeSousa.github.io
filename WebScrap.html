Amazon daily price checker

In this project, we will utilize all the most popular libraries for web scraping. We will first select the webpage and pull the HTML code with requests and then extract all the information we want through Beautifullsoup. The next step will be to save the data into a CSV file. The last part of the project will consist to create a function to scrap the data daily and setting up price surveillance to check when the price will be below a threshold and then sending an alarm to our email.
